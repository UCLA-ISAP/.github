# .github

Organization-wide configuration and documentation for UCLA ISAP's GitHub presence.

## Contents

- **[profile/README.md](profile/README.md)** — Organization profile displayed on the [UCLA-ISAP GitHub page](https://github.com/UCLA-ISAP)
- **[BRANDING.md](BRANDING.md)** — Naming, language, color, and style conventions for ISAP repositories, independent of project.


## About This Repo

This is a [special `.github` repository](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile). GitHub uses it to display the organization profile and store shared community files. It is not a software project.
